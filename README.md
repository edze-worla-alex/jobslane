# JobsLane
Jobslane is a nodejs based job searching platform powered by Artificial Intelligence.

## Getting Started

git clone https://github.com/worlaalex/jobslane.git
cd server
npm install
npm start
### Prerequisites

These are the requirements

```
node-js
npm
body-parser": "^1.19.0",
"cookie-parser": "~1.4.4",
"cors": "^2.8.5",
"debug": "~2.6.9",

```

